# Simple Files Compressor (SFC.exe) 项目说明

---

## 一、项目简介

**SFC.exe** 是一个基于 **Huffman 编码与 AES-128 加密**实现的文件归档工具，支持对**多个文件与目录进行压缩与解压缩**。

该项目主要用于研究**文件系统归档结构、数据压缩以及基础加密实现**。

程序提供**命令行交互（CLI）**，暂未实现图形界面（GUI）。

---

## 二、功能特性

- 支持**多个文件与目录的归档压缩与解压缩**
- 使用 **Huffman 编码**实现数据压缩
- 支持 **AES-128 加密**与 **SHA-256** 密钥哈希
- 采用 **分块压缩/解压机制**（8MB分块），降低内存占用
- 引入 **逻辑根目录（Logical Root）结构**以统一归档路径
- 实现 **路径重复跳过机制**以避免部分重复文件处理
- 支持**长路径**（>260字符）处理

### 5.2 目录扫描算法 (BFS)

EntryProcessor 使用广度优先搜索遍历目录树：

1. 写入**逻辑根节点**，作为所有压缩项的顶层容器
2. 将用户指定的文件/目录加入队列
3. 循环处理队列：
   - 遇到文件：写入文件标准（标志+文件名+大小）
   - 遇到目录：写入目录标准，将子目录加入队列
4. 每处理16KB目录数据，插入一个**分割标准**

### 5.3 二进制编码格式

| 条目类型 | 二进制格式 |
|----------|-----------|
| 文件 | `'1'` + 名称长度(4B) + 名称 + 原始大小(8B) + 压缩大小(8B) |
| 目录 | `'0'` + 名称长度(4B) + 名称 + 子元素数(4B) |
| 逻辑根 | `'3'` + 名称长度(4B) + 名称 + 子元素数(4B) |
| 符号链接 | `'4'` + 链接名长度(4B) + 目标路径长度(4B) + 链接名 + 目标路径 |
| 分割标准 | `'2'` + 块大小(8B) + IV值(16B) |

### 5.4 Huffman压缩流程

每个8MB数据块独立处理：

1. **频率统计** (`statistic_freq`)：统计0-255各字节出现频率
2. **构建最小堆** (`gen_minheap`)：按频率构建优先队列
3. **生成Huffman树** (`gen_hefftree`)：每次合并最小两个节点
4. **生成编码表** (`save_code_inTab`)：遍历树生成各字符编码
5. **序列化树** (`tree_to_plat_uchar`)：前序遍历写入树结构
6. **编码数据** (`encode`)：按编码表转换原始数据

**特殊处理**：当数据块只有一种字符时，Huffman树退化为单节点。

### 5.5 AES加密流程

```cpp
Aes aes(userKey);  // SHA-256哈希取前16字节
aes.doAes(1, inputData, outputData);  // mode=1 加密
aes.doAes(2, inputData, outputData);  // mode=2 解密
```

加密步骤（10轮）：
1. SubBytes - S盒字节替换
2. ShiftRows - 行移位
3. MixColumns - 列混淆（最后一轮跳过）
4. AddRoundKey - 与轮密钥异或

## 六、解压流程详解

### 6.1 整体流程

```
读取 .sy 文件
        │
        ▼
┌─────────────────────────────────┐
│  BinaryStandardLoader           │  读取文件头，验证魔数
│  └─ loadHeaderStandard()        │
└─────────────────────────────────┘
        │
        ▼
┌─────────────────────────────────┐
│  BinaryStandardLoader           │  定位目录块，解密解析
│  └─ headerLoaderIterator()      │  填充文件队列、目录队列
└─────────────────────────────────┘
        │
        ▼
┌─────────────────────────────────┐
│  DecompressionLoop (主循环)      │
│  while (!allLoopIsDone())       │
│  ├─ 创建目录 (directoryQueue)   │
│  ├─ 创建文件 (fileQueue)        │
│  ├─ 读取分割标准                │
│  ├─ 读取加密树 → AES解密        │
│  ├─ Heffman.spawn_tree()        │  恢复Huffman树
│  ├─ 读取加密数据 → AES解密      │
│  ├─ Heffman.decode()            │  Huffman解码
│  └─ DataExporter.export()       │  写入解压数据
└─────────────────────────────────┘
        │
        ▼
    恢复原始文件/目录结构
```

### 6.2 目录块读取与解密

```cpp
void BinaryStandardLoader::headerLoaderIterator(Aes &aes)
{
    // 1. 定位到目录块
    inFile.seekg(header.directoryOffset);

    // 2. 循环读取分割标准
    while (!allDone)
    {
        // 读取分割标准

        // 如果有加密（IV非全0），解密目录块
        if (isEncrypted)
        {
            aes.doAes(2, encryptedBuffer, decryptedBuffer);
        }

        // 解析二进制数据
        parserForLoader->parseEntries(...);
    }
}
```

## 七、.sy 文件格式

### 7.1 整体结构

```
┌─────────────────────────────────────┐
│         文件头 (11字节)              │
│  魔数(4B) | 策略(1B) | 版本(1B)      │
│  头部偏移(1B) | 目录偏移(8B) | 魔数  │
├─────────────────────────────────────┤
│         目录块 (可加密)              │

│          魔数 (4字节)                │
├─────────────────────────────────────┤
│         数据区                       │

├─────────────────────────────────────┤
│          魔数 (4字节)                │
└─────────────────────────────────────┘
```
